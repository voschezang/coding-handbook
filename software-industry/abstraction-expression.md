# Abstraction and Expression

> AI is an expression layer, not an abstraction layer

Innovations can be categorized as abstraction layers or expression layers. See also [systems engineering](systems-engineering.md)



Innovations that have provided new expression layers. These have lowered the barrier to do projects or deploy software. They tend to hide complexity.

- Spreadsheets: everyone can do finance
- Wordpress: everyone can build websites
- Low-code: everyone can code
- Agentic AI: automate everything



Innovations that have provided new abstraction layers.

- Assembly: symbolic representation of hardware
- C: decouple computation from hardware. Portable code
- Object Oriented Programming (OOP): organize code around objects (entities)
- Functional Programming (FP): pure functions, mathematical correctness
- SQL: relational databases and queries
- Virtual Machine (VM): simulate machine as reusable environment, IaC
- Cloud computing: on-demand compute & storage
- APIs: interfaces for delegations and composability. Testable SLAs

Innovations related to technology

- Flowcharts/UML: visual representation of complex logic
- User-stories: describe intent (and outcomes) rather than direct procedures. *"The system shall…"*
- DevOps, CI/CD: shift responsibility and ownership
- Gantt charts, Kanban: visual systems for planning
- KPIs, OKRs: frameworks for metrics
