# Infrastructure

Infrastructure consists of hardware, software, systems and connections.



Abstract components

- Compute
- Storage
- Prediction
- Networking

Network compontens

- Load balancer (LB)
- Router (R)
- Firewall (FW)

Services

- API
- Content delivery network (CDN)

Hardware

-  CPU/GPU
- Memory (RAM)
- Disk (HDD, SSD)
